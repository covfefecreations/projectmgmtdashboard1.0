# Project Dashboard

This is a project dashboard application built with Next.js, React, and TypeScript. It provides a user-friendly interface for managing projects, including project information, timeline, budget, status, and meetings.

## Features

- **Project Information:** View and edit project details, including project name, client, description, and team members.
- **Timeline:** Track project milestones and deliverables.
- **Budget:** Visualize the project budget and spending.
- **Status:** Monitor the overall project status and key metrics.
- **Meetings:** Record and track project meetings.
- **Theme Switching:** Switch between light and dark themes.
- **Responsive Design:** The application is fully responsive and works on all screen sizes.
- **Accessibility:** The application is built with accessibility in mind, including ARIA labels and keyboard navigation.
- **Error Handling:** The application gracefully handles errors and provides a user-friendly fallback UI.

## Getting Started

To get started with the project, follow these steps:

1. **Clone the repository:**

   ```bash
   git clone https://github.com/your-username/project-dashboard.git
   ```

2. **Install the dependencies:**

   ```bash
   pnpm install
   ```

3. **Run the development server:**

   ```bash
   pnpm dev
   ```

4. **Open your browser:**

   Open [http://localhost:3000](http://localhost:3000) to view the application.

## Components

The application is built with a modular component architecture. The main components are:

- **`ProjectInfo`:** Displays and manages project information.
- **`Timeline`:** Displays and manages the project timeline.
- **`Budget`:** Displays and manages the project budget.
- **`Status`:** Displays the project status and key metrics.
- **`Meetings`:** Displays and manages project meetings.
- **`ThemeToggle`:** A button that allows the user to switch between light and dark themes.
- **`ErrorBoundary`:** A component that gracefully handles rendering errors.
